# Google Form Email Automation  

This project connects **Google Forms** with **n8n** to send automatic emails to users. It streamlines communication by integrating form submissions with Gmail, ensuring instant responses and efficient data handling.  

---

## 🚀 Features  
- Sends email automatically after form submission  
- Uses Gmail integration for communication  
- Easy workflow automation with n8n  
- Stores responses in Google Sheets for tracking  

---

## 🛠 Tools Used  
- **n8n** – Workflow automation  
- **Google Forms** – Data collection  
- **Google Sheets** – Response storage  
- **Gmail** – Email notifications  
- **Google Cloud** – Integration support

## 📈 Future Benefits  
- **Scalability**: Can handle thousands of responses without manual effort.  
- **Customization**: Emails can be personalized with names, roles, or feedback details.  
- **Integration**: Can connect with CRMs, HRMS, or other business tools for deeper automation.  
- **Analytics**: Data stored in Sheets can be analyzed for trends, participation rates, and employee engagement.  
- **Time Efficiency**: Reduces repetitive HR tasks like acknowledgment emails or survey confirmations.
